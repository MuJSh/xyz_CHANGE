# XYZ-CHANGE Group Website

This repository contains a clean Jekyll/Liquid academic group website for the XYZ-CHANGE Group.

The site structure is inspired by the academic website template by Spencer H. Bryngelson and follows a data-driven layout:

- Main pages are stored in `_pages/`.
- Group members, publications, news, and conferences are stored in `_data/`.
- Navigation is controlled in `_config.yml`.

## Main pages

- Home
- Group Members
- Publications
- News
- Conferences
- Openings

## Editing guide

- Edit `_config.yml` for site title, subtitle, URL, navigation, and contact information.
- Edit `_data/members.yml` for PI, current members, and alumni.
- Edit `_data/publications.yml` for publications.
- Edit `_data/news.yml` for news items.
- Edit `_data/conferences.yml` for conference records.
- Edit `_pages/openings.md` for recruitment information.

## GitHub Pages
